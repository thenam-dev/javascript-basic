CALL API WITH FETCH

API (Application Programming Interface) là giao diện giúp các ứng dụng giao tiếp với nhau. Trong web, bạn thường gọi API để lấy dữ liệu từ server.
VD: danh sách sản phẩm, thông tin người dùng, bài viết,...

Hiểu đơn giản: API là 1 url (được backend tạo ra), frontend sẽ gọi tới URL đó để lấy dữ liệu (data)
